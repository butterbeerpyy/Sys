# 安装和运行说明

## 方法 1：手动安装依赖（推荐）

### 步骤 1：安装 Python 包
```bash
pip install numpy pillow matplotlib
```

### 步骤 2：运行测试
```bash
cd testmind
python test_pyramid.py
```

## 方法 2：如果没有 pip

### Windows:
```bash
python -m pip install numpy pillow matplotlib
```

### 如果需要国内镜像（更快）:
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple numpy pillow matplotlib
```

## 如果遇到问题

### 问题 1：找不到 Python
**解决**：确保安装了 Python 3.7+
- 下载：https://www.python.org/downloads/

### 问题 2：pip 安装失败
**解决**：尝试使用 conda
```bash
conda install numpy pillow matplotlib
```

### 问题 3：matplotlib 报错
**解决**：设置后端
```bash
export MPLBACKEND=Agg  # Linux/Mac
set MPLBACKEND=Agg     # Windows
```
