# 多分辨率金字塔处理器测试

## 目录结构
```
testmind/
├── test_pyramid.py       # 主测试脚本
├── test_images/          # 测试图像（自动生成）
└── results/              # 测试结果（自动生成）
```

## 快速开始

### 1. 安装依赖
```bash
pip install numpy pillow matplotlib
```

### 2. 运行测试
```bash
cd testmind
python test_pyramid.py
```

### 3. 查看结果
- **终端输出**：详细统计数据
- **results/**：可视化图片

## 测试内容

### ✅ 存储占用测试
- 对比不同 Top-K 值 (20, 40, 60, 80)
- 测量内存占用
- 计算节省比例

### ✅ 精度损失测试
- 计算特征相似度（余弦距离）
- 评估精度损失
- 确定最佳 Top-K

### ✅ 可视化
每张测试图生成：
1. 原始图像
2. 兴趣度热图
3. 选中的区域
4. 统计数据
